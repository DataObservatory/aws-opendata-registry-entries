Receive Data Notifications with Amazon SNS

The following ARN is for an Amazon SNS topic that provides notifications whenever a new ALMA dataset has been added to ALMA on AWS:

arn:aws:sns:TBD:TBD:TBD

This topic publishes an Amazon S3 event message whenever a new ALMA dataset has been created on Amazon S3. It will only accept subscriptions via Amazon SQS or AWS Lambda.
